# 🐄💧 소수 만들기 
## 문제 
[프로그래머스 - 소수 만들기](https://programmers.co.kr/learn/courses/30/lessons/12977)

## 1. 이해
- 주어진 것 : 숫자 배열
- 원하는 것 : 배열 중 3개의 숫자를 뽑아 더한 수가 소수인 경우의 수 

## 2. 계획
- 주어진 숫자 배열에서 3개를 뽑을 수 있는 경우를 다 뽑는다.
- 뽑은 숫자 3개를 더한다. 
- 더한 수가 소수인지 판별한다. 

## 3. 회고 
- 1차 풀이 : 계획에 적은 그대로 반복문을 사용해서 문제를 풀었다. 재귀로 가능할 것 같다. 다음엔 재귀로 풀어보자.
